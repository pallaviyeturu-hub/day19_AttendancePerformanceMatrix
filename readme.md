\# Student Attendance \& Performance Matrix



\## Overview

This C program manages attendance and performance records for 5 students using only loops, 2D arrays, and conditional statements as required. The system accepts student names, a 5x5 attendance matrix, and a 5x5 marks matrix, and generates detailed reports through a menu-driven interface.



\## Features

\- 5x5 Attendance Matrix (1 = Present, 0 = Absent)

\- 5x5 Marks Matrix (0–100 marks)

\- Calculates:

&nbsp; - Total attendance

&nbsp; - Absences per student

&nbsp; - Highest \& lowest attendance

&nbsp; - Total marks

&nbsp; - Average marks

&nbsp; - Class topper and lowest scorer

&nbsp; - Grade assignment

\- Menu-driven interface (no functions used)



\## Constraints Followed

\- No functions  

\- No structures  

\- No pointers  

\- Only loops \& arrays
